Class WMSFacebookDataStoreAssociation

Description
Description | Vars | Methods (details)

Represents two IDs that have been associated.

Represents two IDs that have been associated. This class contains details about an association of two objects, which are represented by their IDs. Other details includes an arbitrary piece of data and a timestamp. The timestamp can be used for any purpose, but the most common is to track when the association was made.
